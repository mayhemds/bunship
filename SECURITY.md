# Security Policy

## Supported Versions

We are currently supporting the following versions of this project with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a potential security hole, please help us keep the community safe by reporting it privately.

### How to Report
Please send an email to **support@bunship.com**. 

To help us act quickly, please include:
* A brief description of the vulnerability.
* Steps to reproduce the issue (proof of concept).
* Any potential impact you’ve identified.

### Our Process
* **Response:** You can expect an initial acknowledgment within **[e.g., 48 hours]**.
* **Fix:** We will work to verify and patch the issue as a priority.
* **Disclosure:** Once the fix is released, we will provide a public advisory and credit you for the discovery.
